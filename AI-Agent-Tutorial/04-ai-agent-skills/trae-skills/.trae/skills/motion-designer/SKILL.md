# Motion Designer

This skill focuses on using motion to communicate, guide, and delight users. Motion is not just decoration; it is functional.

## Core Responsibilities

1.  **Micro-interactions**: Feedback for user actions (e.g., button press, toggle).
2.  **Transitions**: Smoothing the change between states or pages.
3.  **Storytelling**: Guiding the user's attention through a narrative.
4.  **Branding**: Expressing the brand personality through movement.

## Key Concepts

### 1. The 12 Principles of Animation (Disney)
-   **Squash and Stretch**: Giving weight and flexibility.
-   **Anticipation**: Preparing for an action.
-   **Staging**: Directing attention.
-   **Timing**: Speed and rhythm.

### 2. Easing
-   **Linear**: Mechanical and unnatural.
-   **Ease-In/Out**: Natural acceleration and deceleration.
-   **Spring**: Bouncy and playful.

## Workflow

When invoked, this skill will guide you through:

1.  **Concept**: Storyboard the animation.
2.  **Timing**: Define duration and delays.
3.  **Prototyping**: Create high-fidelity motion mockups.
4.  **Handoff**: Provide specs (bezier curves, duration) to developers.

## Usage Scenarios

-   **Feedback**: "The user needs to know the save worked." -> Motion Skill: "Design a checkmark animation that morphs from the save icon."
-   **Navigation**: "Transition between list and detail view." -> Motion Skill: "Use shared element transition to expand the card."
-   **Loading**: "The data takes time to load." -> Motion Skill: "Create a skeleton screen with a shimmering effect."

## Output Format

-   **Animation Spec**: Duration, delay, and easing curve (cubic-bezier).
-   **Lottie JSON**: Vector animation file.
-   **Prototype Video**: Visual demonstration of the motion.
