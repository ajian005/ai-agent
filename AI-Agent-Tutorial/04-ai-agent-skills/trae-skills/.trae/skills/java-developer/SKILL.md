# Java Developer

This skill focuses on building robust, scalable backend applications using Java and the Spring ecosystem.

## Core Responsibilities

1.  **API Development**: Building RESTful and GraphQL APIs.
2.  **Data Access**: interacting with SQL and NoSQL databases.
3.  **System Design**: Implementing microservices and distributed systems.
4.  **Performance**: Tuning the JVM and application code.

## Key Technologies

### 1. Spring Framework
-   **Spring Boot**: Rapid application development.
-   **Spring Data JPA**: Database abstraction.
-   **Spring Security**: Authentication and authorization.

### 2. Java Core
-   **Streams**: Functional data processing.
-   **Concurrency**: Multi-threading and async execution.
-   **JVM**: Garbage collection and memory management.

## Workflow

When invoked, this skill will guide you through:

1.  **Design**: Define the API contract and data model.
2.  **Implementation**: Write Controller, Service, and Repository layers.
3.  **Testing**: Write JUnit and Mockito tests.
4.  **Deployment**: Package as a JAR or Docker image.

## Usage Scenarios

-   **New API**: "Create a user registration endpoint." -> Java Skill: "Define a DTO, validate input, save to DB, and return 201 Created."
-   **Database Query**: "Find users by email." -> Java Skill: "Create a Repository method `findByEmail`."
-   **Async Task**: "Send an email in the background." -> Java Skill: "Use `@Async` and `CompletableFuture`."

## Output Format

-   **Java Class**: Controller, Service, or Entity code.
-   **Configuration**: application.properties/yaml.
-   **Test Code**: Unit test with mocks.
