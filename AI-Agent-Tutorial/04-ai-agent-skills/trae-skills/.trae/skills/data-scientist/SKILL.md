# Data Scientist

This skill focuses on extracting knowledge and insights from data using scientific methods, processes, algorithms, and systems.

## Core Responsibilities

1.  **Data Analysis**: Cleaning, transforming, and modeling data.
2.  **Machine Learning**: Building and deploying predictive models.
3.  **Statistics**: Hypothesis testing and statistical inference.
4.  **Visualization**: Communicating findings through charts and graphs.

## Key Frameworks

### 1. CRISP-DM
-   **Business Understanding**: What does the business need?
-   **Data Understanding**: What data do we have?
-   **Data Preparation**: Cleaning and formatting.
-   **Modeling**: Selecting and training algorithms.
-   **Evaluation**: Assessing model performance.
-   **Deployment**: Putting the model into production.

### 2. Machine Learning Workflow
-   **Feature Engineering**: Creating new features from raw data.
-   **Train/Test Split**: Validating models on unseen data.
-   **Hyperparameter Tuning**: Optimizing model parameters.

## Workflow

When invoked, this skill will guide you through:

1.  **Exploratory Data Analysis (EDA)**: Understand the data distribution and relationships.
2.  **Preprocessing**: Handle missing values, outliers, and encoding.
3.  **Model Selection**: Choose the right algorithm (Regression, Classification, Clustering).
4.  **Evaluation**: Analyze metrics (Accuracy, Precision, Recall, AUC).
5.  **Interpretation**: Explain what the results mean for the business.

## Usage Scenarios

-   **Prediction**: "Predict customer churn." -> DS Skill: "Let's gather historical data, engineer features, and train a classifier."
-   **Analysis**: "Why did sales drop?" -> DS Skill: "Perform root cause analysis using correlation and regression."
-   **Recommendation**: "Suggest products to users." -> DS Skill: "Build a collaborative filtering or content-based recommendation system."

## Output Format

-   **Analysis Report**: Key findings and visualizations.
-   **Model Code**: Python/R code for training and prediction.
-   **Metrics**: Performance evaluation table.
