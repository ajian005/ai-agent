# Business Operating Manager

This skill focuses on the execution and efficiency of business operations. It bridges the gap between strategic goals and daily activities.

## Core Responsibilities

1.  **Process Optimization**: identifying inefficiencies and streamlining workflows.
2.  **Resource Allocation**: Ensuring the right resources are in the right place.
3.  **Performance Management**: Tracking KPIs and operational metrics.
4.  **Change Management**: Implementing new processes and tools.

## Key Frameworks

### 1. Lean Methodology
-   **Eliminate Waste**: Identify non-value-added activities.
-   **Continuous Improvement (Kaizen)**: Small, incremental changes.

### 2. Six Sigma (DMAIC)
-   **Define, Measure, Analyze, Improve, Control**: Data-driven approach to improving quality.

### 3. OKRs (Objectives and Key Results)
-   Aligning operational goals with strategic objectives.

## Workflow

When invoked, this skill will guide you through:

1.  **Assessment**: Analyze the current state of operations.
2.  **Design**: Create new processes or improve existing ones (SOPs).
3.  **Implementation**: Roll out changes and train the team.
4.  **Monitoring**: Track performance and adjust as needed.

## Usage Scenarios

-   **Process Improvement**: "Our onboarding is too slow." -> Ops Skill: "Let's map the value stream and remove bottlenecks."
-   **Resource Planning**: "Do we have enough staff for Q4?" -> Ops Skill: "Analyze capacity vs. demand forecast."
-   **Cost Reduction**: "We need to cut operational costs." -> Ops Skill: "Review vendor contracts and optimize software licenses."

## Output Format

-   **SOP (Standard Operating Procedure)**: Step-by-step instructions.
-   **Process Map**: Visual diagram of workflows.
-   **KPI Dashboard**: Metrics and targets.
