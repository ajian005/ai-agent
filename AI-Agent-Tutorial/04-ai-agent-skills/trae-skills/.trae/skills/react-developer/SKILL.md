# React Developer

This skill specializes in building user interfaces with React, focusing on component architecture, hooks, and state management.

## Core Responsibilities

1.  **Component Design**: Creating reusable, composable UI elements.
2.  **State Management**: Handling local and global application state.
3.  **Performance**: Optimizing rendering and bundle size.
4.  **Testing**: Writing unit and integration tests for components.

## Key Technologies

### 1. React Ecosystem
-   **Hooks**: `useState`, `useEffect`, `useContext`, `useReducer`.
-   **Router**: React Router for navigation.
-   **State**: Redux Toolkit, Zustand, React Query.

### 2. Styling
-   **CSS-in-JS**: Styled-components, Emotion.
-   **Utility**: Tailwind CSS.
-   **Modules**: CSS Modules.

## Workflow

When invoked, this skill will guide you through:

1.  **Structure**: Define the component hierarchy and props.
2.  **Logic**: Implement hooks for state and side effects.
3.  **View**: Write JSX for the UI.
4.  **Optimization**: Memoize expensive computations (`useMemo`, `useCallback`).

## Usage Scenarios

-   **New Component**: "Create a reusable Modal component." -> React Skill: "Use a Portal, manage open state, and handle accessibility (ARIA)."
-   **State Logic**: "Manage a complex form." -> React Skill: "Use `react-hook-form` or a custom hook with validation."
-   **Performance**: "The list is slow to render." -> React Skill: "Implement virtualization with `react-window` and `React.memo`."

## Output Format

-   **Component Code**: JSX/TSX file.
-   **Hook Logic**: Custom hook implementation.
-   **Test Case**: RTL test snippet.
