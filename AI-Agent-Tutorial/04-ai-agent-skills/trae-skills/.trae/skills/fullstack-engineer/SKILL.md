# Fullstack Engineer

This skill enables the development of complete web applications, bridging the gap between client-side interfaces and server-side logic.

## Core Responsibilities

1.  **Frontend**: Building responsive, interactive UIs (React, Vue, etc.).
2.  **Backend**: Developing APIs, business logic, and database interactions.
3.  **Integration**: Connecting the frontend to the backend seamlessly.
4.  **Deployment**: Shipping the full stack to production.

## Key Technologies

### 1. Frontend
-   **Frameworks**: React, Next.js, Vue.
-   **Styling**: Tailwind CSS, CSS Modules.

### 2. Backend
-   **Runtimes**: Node.js, Python, Go.
-   **Databases**: PostgreSQL, MongoDB, Redis.
-   **API**: REST, GraphQL.

## Workflow

When invoked, this skill will guide you through:

1.  **Schema Design**: Define the data model.
2.  **API Development**: Build endpoints to CRUD data.
3.  **UI Implementation**: Create components to display and interact with data.
4.  **Integration**: Connect UI to API.

## Usage Scenarios

-   **Feature**: "Build a comment system." -> Fullstack Skill: "Create `Comment` table, `POST /comments` endpoint, and React form component."
-   **MVP**: "Prototype a todo app." -> Fullstack Skill: "Use Next.js with Prisma and SQLite for a quick fullstack app."
-   **Debug**: "The data isn't showing up." -> Fullstack Skill: "Check the Network tab, API response, and database records."

## Output Format

-   **Fullstack Code**: Frontend component + Backend handler.
-   **Architecture**: Stack diagram.
-   **Deployment Config**: Vercel/Docker config.
