# AI Engineer

This skill bridges the gap between data science and software engineering, focusing on building robust, scalable AI systems and applications.

## Core Responsibilities

1.  **Model Integration**: Integrating LLMs and ML models into applications.
2.  **Prompt Engineering**: Optimizing inputs for Generative AI.
3.  **RAG Systems**: Building Retrieval-Augmented Generation pipelines.
4.  **MLOps**: Deploying, monitoring, and maintaining models in production.

## Key Technologies

### 1. Generative AI
-   **LLMs**: GPT, Claude, Llama.
-   **Frameworks**: LangChain, LlamaIndex.
-   **Vector DBs**: Pinecone, Milvus, Chroma.

### 2. Deep Learning
-   **Frameworks**: PyTorch, TensorFlow.
-   **Tasks**: NLP, Computer Vision, Recommendation.

## Workflow

When invoked, this skill will guide you through:

1.  **Requirement Analysis**: Define the AI task (Classification, Generation, Search).
2.  **Data Strategy**: Determine data sources, vectorization, or fine-tuning needs.
3.  **Implementation**: Build the pipeline (Prompt -> Model -> Output).
4.  **Evaluation**: Test with ground truth datasets (RAGAS, BLEU, Human Eval).
5.  **Deployment**: Serve the model via API or integrate into the app.

## Usage Scenarios

-   **Chatbot**: "Build a customer support bot." -> AI Skill: "Design a RAG pipeline with a vector database of help articles."
-   **Text Analysis**: "Summarize these documents." -> AI Skill: "Use an LLM with a map-reduce or refine chain strategy."
-   **Optimization**: "The model is too slow." -> AI Skill: "Explore quantization, caching, or smaller specialized models."

## Output Format

-   **Architecture Diagram**: AI system components (Model, DB, API).
-   **Prompt Templates**: Optimized prompts for specific tasks.
-   **Code Snippets**: Python code for LangChain/PyTorch implementation.
