# Software Engineering Agent Skills Guide by Role

This guide provides a comprehensive breakdown of skills, tools, and practices for various roles in the software engineering process. It is designed to help AI agents (and humans) understand the responsibilities and capabilities required for each role.

Each role below is linked to its detailed skills guide. Where an executable Agent Skill is available, its ID is noted in parentheses.
