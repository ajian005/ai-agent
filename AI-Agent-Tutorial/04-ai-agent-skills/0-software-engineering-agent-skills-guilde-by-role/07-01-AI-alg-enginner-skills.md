# AI Algorithm Engineer Skills

The AI Algorithm Engineer focuses on the mathematical and theoretical foundations of AI models, designing new algorithms or optimizing existing ones for specific tasks.

## Role Definition
Focuses on deep learning architectures, mathematical optimization, research reproduction, and algorithmic efficiency.

## Skills & Tools

### 1. Deep Learning Architectures
Designing neural networks.

- **Tool: PyTorch / TensorFlow**
  - *Description*: Core frameworks for building models.
  - *Practice*: Implement custom layers, loss functions, and optimizers.
  - *Agent Prompt*: "Implement a custom Attention mechanism layer in PyTorch."

- **Tool: Neural Architecture Search (NAS)**
  - *Description*: Automating the design of ANNs.
  - *Practice*: Use tools or write scripts to find optimal architectures.
  - *Agent Prompt*: "Explain the principles of EfficientNet and scaling laws."

### 2. Mathematical Optimization
Improving model convergence and performance.

- **Tool: Optimization Algorithms**
  - *Description*: SGD, Adam, AdamW, LAMB.
  - *Practice*: Tune learning rates, schedulers, and regularization (Dropout, Weight Decay).
  - *Agent Prompt*: "Compare AdamW vs. SGD with momentum for training Transformers."

- **Tool: Loss Functions**
  - *Description*: Defining the objective.
  - *Practice*: Design custom loss functions for multi-task learning or specific constraints.
  - *Agent Prompt*: "Implement a Focal Loss function for imbalanced classification."

### 3. Research & Reproduction
Staying at the cutting edge.

- **Tool: arXiv / Papers with Code**
  - *Description*: Sources for the latest research.
  - *Practice*: Read papers and implement them in code.
  - *Agent Prompt*: "Summarize the key innovations in the 'Attention Is All You Need' paper."

- **Tool: Benchmarking**
  - *Description*: Standard datasets (ImageNet, GLUE, SQuAD).
  - *Practice*: Evaluate models against SOTA (State of the Art).
  - *Agent Prompt*: "Set up a benchmark script to evaluate a model on the GLUE dataset."

### 4. Efficient AI
Making models faster and smaller.

- **Tool: Model Compression**
  - *Description*: Techniques to reduce model size.
  - *Practice*: Pruning, Quantization (INT8, FP16), Distillation.
  - *Agent Prompt*: "Apply dynamic quantization to a PyTorch model for CPU inference."

- **Tool: High-Performance Computing**
  - *Description*: CUDA, TensorRT.
  - *Practice*: Optimize kernels for GPU execution.
  - *Agent Prompt*: "Optimize a transformer block using TensorRT."
