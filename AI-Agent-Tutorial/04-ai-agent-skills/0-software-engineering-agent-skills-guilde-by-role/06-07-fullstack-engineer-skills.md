# Fullstack Engineer Skills

The Fullstack Engineer is versatile, capable of handling both frontend and backend development. They understand the entire web development stack, from the user interface to the database.

## Role Definition
Focuses on end-to-end feature delivery, API integration, database design, and frontend implementation.

## Skills & Tools

### 1. Frontend Development
Building the client-side application.

- **Tool: React / Vue / Angular**
  - *Description*: JavaScript frameworks for building UIs.
  - *Practice*: Create responsive and interactive components.
  - *Agent Prompt*: "Build a React component for a user profile page that fetches data from an API."

- **Tool: CSS Frameworks**
  - *Description*: Tailwind CSS, Bootstrap, Material UI.
  - *Practice*: Style applications efficiently.
  - *Agent Prompt*: "Style the login form using Tailwind CSS."

### 2. Backend Development
Building the server-side logic.

- **Tool: Node.js / Python / Go / Java**
  - *Description*: Server-side runtimes and languages.
  - *Practice*: Build RESTful or GraphQL APIs.
  - *Agent Prompt*: "Create an Express.js route to handle user registration."

- **Tool: Authentication**
  - *Description*: JWT, OAuth, Session management.
  - *Practice*: Secure endpoints and manage user sessions.
  - *Agent Prompt*: "Implement JWT authentication middleware for a Node.js API."

### 3. Database Management
Designing and querying databases.

- **Tool: SQL / NoSQL**
  - *Description*: PostgreSQL, MongoDB, Redis.
  - *Practice*: Design schemas and write efficient queries.
  - *Agent Prompt*: "Design a MongoDB schema for a blog post with comments."

- **Tool: ORM / ODM**
  - *Description*: Prisma, TypeORM, Mongoose.
  - *Practice*: Interact with the database using code.
  - *Agent Prompt*: "Write a Prisma query to find all users who signed up in the last week."

### 4. DevOps & Deployment
Deploying and managing the application.

- **Tool: Docker**
  - *Description*: Containerization platform.
  - *Practice*: Create Dockerfiles and docker-compose configurations.
  - *Agent Prompt*: "Create a docker-compose file for a PERN stack (Postgres, Express, React, Node)."

- **Tool: Cloud Platforms**
  - *Description*: AWS, Vercel, Netlify, Heroku.
  - *Practice*: Deploy frontend and backend services.
  - *Agent Prompt*: "Deploy the Next.js frontend to Vercel and the backend to Railway."
