# Frontend Engineer Skills

The Frontend Engineer is responsible for implementing the user interface and user experience of web applications, ensuring responsiveness, performance, and accessibility.

## Role Definition
Focuses on converting designs into code, client-side logic, component reuse, and browser compatibility.

## Skills & Tools

### 1. UI Implementation
Building pixel-perfect interfaces from designs.

- **Tool: Component Library System**
  - *Description*: Creating and using reusable UI components (e.g., Button, Card, Modal).
  - *Practice*: Implement atomic design principles.
  - *Agent Prompt*: "Create a React component for a reusable 'User Card'."

- **Tool: CSS Frameworks**
  - *Description*: Libraries for styling (Tailwind CSS, Bootstrap, Material UI).
  - *Practice*: Apply consistent styling and responsive layouts.
  - *Agent Prompt*: "Style a login form using Tailwind CSS with responsive breakpoints."

### 2. Client-Side Logic & State Management
Handling user interactions and data flow.

- **Tool: State Management Libraries**
  - *Description*: Managing application state (Redux, Zustand, Context API).
  - *Practice*: Define actions, reducers, and store structure.
  - *Agent Prompt*: "Design a Redux slice for managing the 'Shopping Cart' state."

- **Tool: Data Fetching**
  - *Description*: Retrieving data from APIs (Axios, Fetch, React Query).
  - *Practice*: Handle loading states, errors, and caching.
  - *Agent Prompt*: "Implement a data fetching hook for the 'Product List' API using React Query."

### 3. Performance Optimization
Ensuring the application runs smoothly.

- **Tool: Lighthouse**
  - *Description*: Automated tool for improving the quality of web pages.
  - *Practice*: Audit performance, accessibility, and SEO.
  - *Agent Prompt*: "Analyze the Lighthouse report and suggest optimizations for LCP (Largest Contentful Paint)."

- **Tool: Bundle Analyzer**
  - *Description*: Visualizing the size of webpack output files.
  - *Practice*: Identify large dependencies and implement code splitting.
  - *Agent Prompt*: "Analyze the bundle size and suggest libraries to lazy load."

### 4. Testing & Quality
Ensuring code reliability.

- **Tool: Jest / Vitest**
  - *Description*: JavaScript testing frameworks.
  - *Practice*: Write unit tests for utility functions and components.
  - *Agent Prompt*: "Write a unit test for the 'formatDate' utility function."

- **Tool: React Testing Library**
  - *Description*: Testing utilities for React DOM.
  - *Practice*: Test components from the user's perspective (e.g., clicking buttons).
  - *Agent Prompt*: "Write a test case to verify the 'Submit' button is disabled when input is empty."
