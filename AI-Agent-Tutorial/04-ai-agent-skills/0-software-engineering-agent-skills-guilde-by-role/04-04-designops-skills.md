# DesignOps Skills

DesignOps (Design Operations) focuses on the operationalization of design processes, optimizing workflows, and enabling design teams to scale efficiently.

## Role Definition
Focuses on process, tools, culture, hiring, and bridging the gap between design and engineering.

## Skills & Tools

### 1. Workflow Optimization
Streamlining how design work gets done.

- **Tool: Design Process Mapping**
  - *Description*: Visualizing the end-to-end design workflow.
  - *Practice*: Identify bottlenecks and handoff friction points.
  - *Agent Prompt*: "Map the current design-to-development handoff process."

- **Tool: Task Management (Jira/Asana for Design)**
  - *Description*: Managing design tickets and backlogs.
  - *Practice*: Integrate design sprints with engineering sprints.
  - *Agent Prompt*: "Set up a Jira workflow for design requests."

### 2. Tooling & Infrastructure
Managing the design tech stack.

- **Tool: Asset Management (DAM)**
  - *Description*: Organizing and storing design assets.
  - *Practice*: Ensure easy access and version control (Abstract, Figma branching).
  - *Agent Prompt*: "Evaluate tools for version controlling Sketch/Figma files."

- **Tool: Design System Governance**
  - *Description*: Processes for maintaining and updating the Design System.
  - *Practice*: Define how new components are proposed and approved.
  - *Agent Prompt*: "Draft a governance model for the Design System contributions."

### 3. Team Management & Culture
Scaling the team effectively.

- **Tool: Onboarding Playbooks**
  - *Description*: Standardized process for new designers.
  - *Practice*: Reduce time-to-productivity for new hires.
  - *Agent Prompt*: "Create an onboarding checklist for a new Product Designer."

- **Tool: Skills Matrix**
  - *Description*: Mapping team capabilities.
  - *Practice*: Identify skill gaps and plan training.
  - *Agent Prompt*: "Create a skills matrix for the UX research team."

### 4. Design Quality & Metrics
Measuring the impact of design.

- **Tool: Design QA Process**
  - *Description*: Ensuring implemented code matches design specs.
  - *Practice*: Establish a review step before code merge.
  - *Agent Prompt*: "Define a Design QA checklist for frontend implementation."

- **Tool: Design Metrics**
  - *Description*: KPIs for the design team (e.g., velocity, satisfaction).
  - *Practice*: Track the efficiency and quality of design output.
  - *Agent Prompt*: "Identify 3 KPIs to measure DesignOps success."
