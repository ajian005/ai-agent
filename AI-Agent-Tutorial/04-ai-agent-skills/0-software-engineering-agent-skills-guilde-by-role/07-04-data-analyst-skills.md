# Data Analyst Skills

The Data Analyst interprets data to help the organization make better decisions. They focus on descriptive and diagnostic analytics.

## Role Definition
Focuses on SQL querying, data visualization, reporting, and communicating insights.

## Skills & Tools

### 1. Data Querying & Manipulation
Getting the data.

- **Tool: Advanced SQL**
  - *Description*: The language of databases.
  - *Practice*: Use Window Functions, CTEs (Common Table Expressions), and complex Joins.
  - *Agent Prompt*: "Write a SQL query using a Window Function to calculate a running total."

- **Tool: Excel / Google Sheets**
  - *Description*: Spreadsheet software.
  - *Practice*: Use VLOOKUP, Pivot Tables, and Power Query for quick analysis.
  - *Agent Prompt*: "Explain how to use a Pivot Table to summarize sales by region."

### 2. Data Visualization & Reporting
Showing the data.

- **Tool: Tableau / Power BI**
  - *Description*: Business Intelligence (BI) platforms.
  - *Practice*: Build interactive dashboards and reports.
  - *Agent Prompt*: "Design a Power BI dashboard for tracking monthly recurring revenue (MRR)."

- **Tool: Looker / Metabase**
  - *Description*: Modern BI tools.
  - *Practice*: Define metrics and explore data self-service.
  - *Agent Prompt*: "Create a LookML view for the 'Orders' table."

### 3. Statistical Analysis
Understanding the data.

- **Tool: A/B Testing**
  - *Description*: Comparing two versions to see which performs better.
  - *Practice*: Calculate sample size, significance, and confidence intervals.
  - *Agent Prompt*: "Calculate the statistical significance of an A/B test result."

- **Tool: Descriptive Statistics**
  - *Description*: Mean, Median, Mode, Standard Deviation.
  - *Practice*: Summarize dataset characteristics.
  - *Agent Prompt*: "Calculate the standard deviation of this dataset to understand variability."

### 4. Communication
Telling the story.

- **Tool: Data Storytelling**
  - *Description*: Combining data, visuals, and narrative.
  - *Practice*: Present findings to non-technical stakeholders.
  - *Agent Prompt*: "Draft a presentation summarizing the impact of the new marketing campaign."
