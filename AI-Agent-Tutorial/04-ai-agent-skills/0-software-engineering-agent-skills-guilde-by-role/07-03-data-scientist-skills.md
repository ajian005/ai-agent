# Data Scientist Skills

The Data Scientist is responsible for extracting insights from data, building predictive models, and applying advanced analytics to solve business problems.

## Role Definition
Focuses on statistics, machine learning, data visualization, and exploratory data analysis.

## Skills & Tools

### 1. Exploratory Data Analysis (EDA)
Understanding the data before modeling.

- **Tool: Pandas / NumPy**
  - *Description*: Python libraries for data manipulation and analysis.
  - *Practice*: Clean, transform, and summarize data.
  - *Agent Prompt*: "Perform EDA on the 'Customer Churn' dataset using Pandas."

- **Tool: Data Visualization**
  - *Description*: Creating charts and graphs (Matplotlib, Seaborn, Plotly).
  - *Practice*: Identify patterns, trends, and outliers.
  - *Agent Prompt*: "Visualize the distribution of 'Age' vs. 'Spending Score'."

### 2. Machine Learning
Building predictive models.

- **Tool: Scikit-learn**
  - *Description*: Machine learning library for Python.
  - *Practice*: Implement algorithms like Regression, Classification, Clustering.
  - *Agent Prompt*: "Train a Random Forest classifier to predict customer churn."

- **Tool: Deep Learning Frameworks**
  - *Description*: Libraries for neural networks (TensorFlow, PyTorch, Keras).
  - *Practice*: Build models for image recognition, NLP, etc.
  - *Agent Prompt*: "Build a CNN model for image classification using PyTorch."

- **Tool: Model Evaluation**
  - *Description*: Assessing model performance.
  - *Practice*: Use metrics like Accuracy, Precision, Recall, F1-Score, ROC-AUC.
  - *Agent Prompt*: "Evaluate the model performance using a confusion matrix."

### 3. Feature Engineering
Creating new features to improve model performance.

- **Tool: Feature Selection**
  - *Description*: Selecting the most relevant variables.
  - *Practice*: Reduce dimensionality and prevent overfitting.
  - *Agent Prompt*: "Perform feature selection using Recursive Feature Elimination (RFE)."

- **Tool: Data Preprocessing**
  - *Description*: Preparing data for modeling.
  - *Practice*: Handle missing values, encode categorical variables, scale features.
  - *Agent Prompt*: "Preprocess the dataset by handling missing values and one-hot encoding."

### 4. Communication & Storytelling
Presenting findings to stakeholders.

- **Tool: Jupyter Notebooks**
  - *Description*: Interactive computing environment.
  - *Practice*: Document the analysis process and results.
  - *Agent Prompt*: "Create a Jupyter Notebook summarizing the analysis findings."

- **Tool: Dashboarding**
  - *Description*: Creating interactive dashboards (Tableau, Power BI, Streamlit).
  - *Practice*: Enable stakeholders to explore data.
  - *Agent Prompt*: "Build a Streamlit app to showcase the sales forecast model."
