# Endstation (Backend) Engineer Skills

The Endstation (Backend) Engineer is responsible for server-side logic, database interactions, API development, and ensuring system stability and performance.

## Role Definition
Focuses on data processing, business logic implementation, API design, and system integration.

## Skills & Tools

### 1. API Development
Building robust and secure interfaces.

- **Tool: Web Frameworks**
  - *Description*: Libraries for building web applications (Spring Boot, Django, Gin, Express).
  - *Practice*: Implement RESTful or GraphQL endpoints.
  - *Agent Prompt*: "Create a REST API endpoint for 'Create Order' using Spring Boot."

- **Tool: API Documentation**
  - *Description*: Tools for documenting APIs (Swagger/OpenAPI, Postman).
  - *Practice*: Keep documentation in sync with code.
  - *Agent Prompt*: "Generate Swagger documentation for the 'User Management' controller."

### 2. Database Management
Designing and optimizing data storage.

- **Tool: SQL / NoSQL**
  - *Description*: Query languages and database systems (MySQL, PostgreSQL, MongoDB, Redis).
  - *Practice*: Write efficient queries, design schemas, and manage indexes.
  - *Agent Prompt*: "Write an SQL query to find the top 10 spending users in the last month."

- **Tool: ORM (Object-Relational Mapping)**
  - *Description*: Converting data between incompatible type systems (Mybatis, Hibernate, SQLAlchemy, GORM).
  - *Practice*: Map database tables to code objects.
  - *Agent Prompt*: "Define a User entity and repository using JPA/Hibernate."

### 3. System Architecture & Patterns
Writing maintainable and scalable code.

- **Tool: Design Patterns**
  - *Description*: Reusable solutions to common problems (Singleton, Factory, Strategy).
  - *Practice*: Apply appropriate patterns to solve design challenges.
  - *Agent Prompt*: "Refactor this payment processing code using the Strategy Pattern."

- **Tool: Microservices / Monolith**
  - *Description*: Architectural styles.
  - *Practice*: Decompose complex systems into manageable services.
  - *Agent Prompt*: "Design the service boundaries for an 'Order Processing' microservice."

### 4. Security & Performance
Protecting data and ensuring speed.

- **Tool: Authentication & Authorization**
  - *Description*: Verifying identity and access rights (OAuth2, JWT).
  - *Practice*: Implement secure login and role-based access control (RBAC).
  - *Agent Prompt*: "Implement JWT authentication filter for the API gateway."

- **Tool: Profiling & Monitoring**
  - *Description*: Tools to analyze performance (Prometheus, Grafana, pprof, JProfiler).
  - *Practice*: Identify CPU/Memory hotspots and optimize.
  - *Agent Prompt*: "Analyze the CPU profile to identify the cause of high latency."
