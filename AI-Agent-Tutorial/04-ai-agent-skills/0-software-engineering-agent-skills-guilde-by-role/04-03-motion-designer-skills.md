# Motion Designer Skills

The Motion Designer is responsible for bringing interfaces to life through animation, transitions, and micro-interactions.

## Role Definition
Focuses on animation, timing, easing, storytelling, and enhancing usability through motion.

## Skills & Tools

### 1. Animation Principles
Applying the 12 principles of animation to UI.

- **Tool: Easing Functions**
  - *Description*: Controlling the rate of change of a parameter over time (Linear, Ease-In, Ease-Out, Spring).
  - *Practice*: Make movements feel natural and physical.
  - *Agent Prompt*: "Suggest appropriate easing curves for a modal entrance."

- **Tool: Timing & Duration**
  - *Description*: How long an animation takes.
  - *Practice*: Ensure animations are fast enough to not block the user (200-500ms).
  - *Agent Prompt*: "Define the duration guidelines for hover effects vs. page transitions."

### 2. Micro-interactions
Providing feedback and delight.

- **Tool: Lottie / Rive**
  - *Description*: Formats for rendering vector animations on the web/mobile.
  - *Practice*: Create complex animations (e.g., success checkmark, loading state) that are lightweight.
  - *Agent Prompt*: "Create a JSON animation for a 'Success' state using Lottie."

- **Tool: State Transitions**
  - *Description*: Animating changes between component states (e.g., toggle switch).
  - *Practice*: Clarify what is happening (morphing, expanding).
  - *Agent Prompt*: "Design the transition for a card expanding into a detail view."

### 3. Prototyping Motion
Visualizing animations before code.

- **Tool: After Effects / Principle / ProtoPie**
  - *Description*: Tools for creating high-fidelity motion prototypes.
  - *Practice*: Demonstrate exact behavior to developers.
  - *Agent Prompt*: "Describe the workflow for prototyping a swipe gesture in ProtoPie."

### 4. Implementation
Working with developers to ship motion.

- **Tool: CSS Animations / Framer Motion**
  - *Description*: Libraries and standards for coding animations.
  - *Practice*: Implement performant animations (using transform/opacity).
  - *Agent Prompt*: "Write CSS keyframes for a pulsing notification dot."

- **Tool: Motion System**
  - *Description*: Documenting motion values in the Design System.
  - *Practice*: Standardize duration tokens and easing curves.
  - *Agent Prompt*: "Define a Motion System with standardized tokens for 'Fast', 'Normal', and 'Slow'."
